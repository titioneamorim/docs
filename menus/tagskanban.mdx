Olá, tudo bem? Neste guia, vamos aprender como configurar as tags e o kanban. Vamos lá!

## Configurando Tags

Primeiro, clique no menu **"Tags"** e adicione uma nova tag.

### Adicionando Exemplos de Tags

- Vou colocar como exemplo a tag **"Clientes"** e escolher uma cor verde. Adicionado!
- Agora, uma nova tag para **"Leads Quentes"** com uma cor vermelha e outra para **"Leads Frios"** com uma cor azul. Adicionadas!
- Para exemplificar, adicionaremos uma tag para **"Parceiros"** com uma cor laranja. Feito!

## Associando Tags ao Kanban

As tags estão, de certa forma, ligadas ao kanban. Se você acessar o kanban, verá que não há nenhum kanban vinculado a uma tag específica.

### Vinculando Tags ao Kanban

- Volte para as tags e associe, por exemplo, a tag **"Clientes"** a um kanban e salve.
- Ao retornar ao kanban verá os clientes específicos que foram marcados.
- Faça o mesmo com a tag **"Leads Quentes"** e **"Leads Frios"**. Ao marcá-los e salvar, ao voltar ao kanban veremos os clientes marcados como **"Leads Quentes"** e **"Leads Frios"**.

Por exemplo, se você tiver uma tag para **"Parceiros"** que não está associada a nenhum kanban, ela aparecerá apenas como marcação. Isso é útil para indicar parcerias mesmo que não precise ser exibida no kanban.

## Conclusão

É assim que configuramos as tags e o kanban no atendimento. Ao adicionar a tag, ela aparecerá no kanban se configurado, caso contrário, ficará apenas como marcação.
